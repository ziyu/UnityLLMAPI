# Changelog
本文档记录了此项目的所有重要变更。

格式基于[Keep a Changelog](http://keepachangelog.com/en/1.0.0/)，
版本号遵循[Semantic Versioning](http://semver.org/spec/v2.0.0.html)。

## [1.1.0] - 2024-11-10
### 新增
- ChatBot实现
  - ChatbotService封装
  - 对话历史管理
  - 系统提示词配置
  - 灵活的配置选项
- Tool Calling功能支持
  - 支持定义和执行自定义工具
  - 工具调用权限控制
  - 工具执行结果处理
- 流式响应支持
  - 实时获取API响应内容
  - 支持UI实时更新
  - 支持取消操作

### 优化
- 重构代码结构，提高可维护性
  - 分离服务层和工具层
  - 优化接口设计
  - 改进异常处理机制
- 改进错误处理
  - 更详细的错误信息
  - 分类异常处理
  - 完善日志系统

### 修复
- 修复长文本响应可能的截断问题
- 修复并发请求的潜在问题
- 修复JSON解析的边界情况
